% Thesis Abstract --------------------------------------------

\begin{abstract}
\hrule \bigskip \vspace*{1cm}
El siguiente trabajo analiza la desambiguación en lenguaje natural revisando la diferente literatura donde encontramos ejemplos de tareas lingüísticas complejas como traducciones, resúmenes de textos, recuperación de información, clasificación

Para justificar nuestro objetivo, hicimos una muestra con un ejemplo de clasificación usando el método KNN para clasificar tipos de textos.
\end{abstract}
